Bootcamp 2020: Aim trainer

My first programming project, concluding the 2 week Programming Bootcamp at City (2021). A simple target shooting game where I used a class to produce an object with random properties. It implements counters, mouseclick detection, and conditional statements to keep the game going indefinitely. I originally planned to add an array to create multiple objects but the syntax was proving to be too challenging at my current stage of experience.

Check out the code on Github: https://github.com/AbyanIsmangil/Simple-Aim-Trainer
